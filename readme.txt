== Description ==

A simple WordPress plugin that helps proteting the site's content from humans and bots.

There are some protections available:

* Disable Contextual Menu
* Disable Text Selection
* Disable Feed
* Disable Iframe Embedding
* Protect REST API

Admin settings are available in Settings/Content Protection.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `protect-my-content` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
